## About the Project
This project focuses on predicting whether a passenger survived the Titanic disaster using Machine Learning techniques.

The Titanic dataset is widely used for learning Machine Learning because it contains real-world passenger data such as age, gender, passenger class, fare, and embarkation point.  
Using this dataset, different Machine Learning models are trained to predict survival outcomes.

---

## Objectives of the Project
The main objectives of this project are:

- To understand and analyze real-world datasets
- To perform data cleaning and preprocessing
- To apply Machine Learning algorithms for classification
- To compare model performance
- To visualize patterns and results using graphs

---

## Dataset
- Dataset Name: Titanic Dataset
- Source: Kaggle Titanic Dataset
- File Used: `titanic.csv`

### Important Features:
- PassengerId
- Pclass
- Sex
- Age
- SibSp
- Parch
- Fare
- Embarked
- Survived (Target variable)

## Machine Learning Models Used
- Logistic Regression
- Decision Tree Classifier
- K-Means Clustering (for pattern analysis)
- PCA (Principal Component Analysis) for visualization

---

## Results
- Data cleaning improved overall model performance
- Decision Tree performed better than Linear models
- Clustering helped identify hidden passenger groups
- PCA helped visualize clusters effectively

---

## Conclusion
This project demonstrates the complete Machine Learning pipeline starting from data preprocessing to model evaluation.

It helps beginners understand how Machine Learning works on real-world data and how different models perform on the same dataset.
